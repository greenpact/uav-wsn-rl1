OMNeT++ SCENARIO CONFIGURATION
OFFLINE RL TRAINING DATA GENERATOR
FOR UAV–ASSISTED WIRELESS SENSOR NETWORKS

------------------------------------------------------------

1. PURPOSE OF THESE SCENARIOS

These simulation scenarios are designed ONLY for generating
training datasets for offline Deep Q Network (DQN) training.

The objectives are:

• generate diverse routing states
• produce balanced state-action coverage
• avoid dataset bias
• represent realistic UAV–WSN operating conditions

The simulator acts as a digital twin of the network to
produce RL samples:

(s, a, r, s')

The scenarios emphasize ENVIRONMENT DIVERSITY rather
than protocol comparison.

------------------------------------------------------------

------------------------------------------------------------

15. REWARD FUNCTION USED IN DATASET

Reward formulation:

r = w1*Ds − w2*Ec − w3*Td − w4*Pl

Where

Ds = successful packet delivery
Ec = energy consumption
Td = delay
Pl = packet loss

Recommended weights:

w1 = 5
w2 = 1
w3 = 2
w4 = 5

For each simulation episode:

1. Randomly select network area
2. Randomly deploy nodes
3. Randomly select traffic model
4. Randomly select UAV trajectory
5. Randomly select behavior policy
6. Run simulation
