COMPREHENSIVE OMNET++ SIMULATION SCENARIO DESIGN
FOR UAV–WSN ROUTING WITH OFFLINE RL DATASET GENERATION

1. PURPOSE
This document defines a scientifically justified and statistically valid
set of simulation scenarios for generating reinforcement learning datasets
and evaluating UAV-assisted Wireless Sensor Network routing protocols.

The scenarios follow commonly accepted parameter ranges used in
WSN, UAV networking, and reinforcement learning research.

The configuration ensures:
- realistic network behavior
- statistically valid experiments
- sufficient dataset diversity for RL training
- reproducible experimental methodology

------------------------------------------------------------

16. REWARD FUNCTION

Reward formulation:

r = w1*Ds - w2*Ec - w3*Td - w4*Pl

Where

Ds = successful packet delivery
Ec = energy consumption
Td = delay
Pl = packet loss

Recommended weights:

w1 = 5
w2 = 1
w3 = 2
w4 = 5
